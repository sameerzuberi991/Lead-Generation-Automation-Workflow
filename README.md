# Lead-Generation-Automation-Workflow
This repository contains an n8n automation workflow that generates fake but realistic B2B leads using AI and stores them directly into Google Sheets.
It is useful for testing CRM pipelines, demos, cold-email systems, or sales automation workflows without using real data.
